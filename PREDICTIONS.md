1. Double: IThis function will take the user input (n), add it to itself, and return the result.

2. Square: This function will square n, it is quite simple as it just returns n * n.

3. IsEven: This function checks if n % 2 is equal to 0, if it is, then it returns "(n) is even", if it is odd, it returns "(n) is not even".

4. IsOdd: Works similar to isEven, it checks if n % 2 is greater than 0, if it is, it returns "(n) is odd", if it isn't, it returns "(n) is not odd".

5. Power: It takes two inputs, (n) and (n2), it then calculates (n) to the power of (n2), and returns the output.

6. Multiply: It takes two inputs, (n) and (n2), calculates n * n2, then returns the output.
